# Figma Design

Local recreation of the public Figma Sites design:

https://www.figma.com/site/r99u5Pad24nAG9PDffEtps/my-website?t=3I8Kydck8rMOoXP6-0

Open `index.html` in a browser to view the static implementation.

The downloaded `thumbnail.png` and `thumbnail-large.png` files are kept as
visual references from the Figma source. Cropped image assets live in `assets/`.

## Todo

The Figma Make Todo design is included at:

https://www.figma.com/make/CaIPDqlCAGopPYFQCWa629/Todo?t=3I8Kydck8rMOoXP6-0

Open `todo.html` to view the local page with the Figma Make embed.
