# Alternate Todo App Design

## Figma File

- Name: `Todo app - alternate design`
- URL: https://www.figma.com/design/wfKPMeZSzcGLHIMruoP44N
- Page: `Todo Alt Design`
- Primary frame: `Todo App Alt Concept`

## Purpose

This is an alternate visual design for the Todo app that keeps the same product flow as the original version:

1. Add a task
2. Select a date
3. Choose a priority
4. Review the active task list
5. Track completed items separately

## Visual Direction

- Warm paper-toned background instead of the original cool blue gradient
- Editorial dashboard feel with stronger section hierarchy
- Dark summary card for quick daily status
- Rounded form surfaces with softer borders
- Priority chips for `High`, `Medium`, and `Low`
- Dedicated completed section to keep progress visible without cluttering the active queue

## Main Sections

- Hero area with title and summary card
- Left composer panel:
  - task input
  - date input
  - priority selector
  - add-task action
- Right task board:
  - task overview chips
  - active priority queue
  - completed section

## Notes

- The design was created directly in Figma through MCP.
- This folder currently stores the design reference and handoff details because the repository does not contain a matching frontend app to implement the screen as code.
- If needed, the next step can be to build this design as HTML/CSS/React in a separate app folder.
